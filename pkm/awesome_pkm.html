**Awesome pkm**

[abstractsun](../): On this page, I list various tools and techniques *roughly* related to "personal knowledge management" (pkm), notetaking, task management, and time management that I either personally use, OR I could see myself using if my needs were different.

The name is a reference to the "awesome" lists found on code forges.

**Tool name** |   | **Description**
-|-|-
[Super Productivity](https://super-productivity.com/) | 🚲🌐 | 
A user-friendly time-tracking and task management tool.

[Frogtab](https://frogtab.com/) | 🚲🐦🌐🌐 | 
The simple text interface allows for quickly jotting down tasks. Tasks can be postponed, or automatically added based on the day of the week. Sending a note from another device requires an internet connection.

[GTG](https://github.com/getting-things-gnome/gtg) (Linux-only) | 🚲 | 
An elegant task tracker for Linux desktop, also known as "Getting Things GNOME!" It has a task focus view, and tasks can have deadlines with granularity of one day.

Email | 🚲🌐🌐 | 
A useful way to send notes to another computer.

Graphical calendar | 🚲 | 
Some notes and tasks work best with a deadline. For advanced use, consider looking for one with iCalendar import/export.

Instant messaging | 🚲🌐🌐 | 
Rather than trying to move a note between applications, just use the tool at hand.

[Krita](https://krita.org/) | 🚲🎨 | 
The canvas can be expanded in any direction, allowing for unrestricted doodles and notes

Password manager | 🚲 | 
Apparently, some people use a password manager to store important notes. Weird, but makes sense! Depending on the password manager, this may give you encryption, syncing, and/or version history, without any additional effort.

One Big Text File | 🏃🐦 | 
Not everyone needs a fancy system. With a good text editor, your text file can be tens of thousands of lines long without any issues, and can be easily synced.

[Markdeep](https://casual-effects.com/markdeep/) | 🔧🌐 | 
Allows an HTML document to be written inside of as if it is a markdown file.

[calendar.txt](https://terokarvinen.com/2021/calendar-txt/) | 🔧🐦 | 
You don't need to use fancy scripts or follow fancy rules to have an agenda or diary in plaintext. Just download the calendar.txt template and edit in your text editor of choice. (but you can make it fancy if you want to)

[vim](https://www.vim.org/) | 🔧🔧🐦 | 
An opinionated hacker's text editor.

[braindump](../braindump) | 🔧🔧🐦➕ | 
My collection of notetaking scripts run from the command line, with a focus on easy file naming and note review.

[Denote](https://protesilaos.com/emacs/denote) for Emacs | 🔧🔧🏃🐦 | 
Denote's clever file naming scheme allows for title, timestamp, and tags to coexist within a file name. More generally, Denote is a tool for managing a notebook in Emacs.

[Fossil](http://fossil-scm.org/) | 🔧🔧🌐 | 
Fossil is a version control system. Its built-in issue tracker can have custom views scripted with SQLite.

[pandoc](https://pandoc.org/) | 🔧🔧🏃 | 
A swiss army knife for plaintext markup conversion, for example converting html back to markdown.

qrencode | 🔧🔧🏃🐦 | 
A command line tool for creating QR codes. Use `base64` to encode special characters. Graphical QR code reader and generator software also exist on many platforms.

grep | 🔧🔧🏃🐦 | 
A collection of plaintext notes on a hard drive can be easily searched with many tools. grep stands out due to very fast speed, support for regular expressions, the ability to add lines of context to search results, and other customization features. See also: `git-grep` and `ack`, which automatically exclude some files from search by default.
